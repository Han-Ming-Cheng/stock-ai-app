📈 美股 AI 財報 & 技術分析工具（Gemini 版）

這是一個用 Streamlit + Google Gemini 打造的美股分析小工具。
你只要輸入股票代號（例：AAPL, TSLA, NVDA），系統就會自動幫你：

抓 Yahoo Finance 的股價 & 基本面

顯示專業版 K 線圖 + MA 線 + RSI / 成交量

彙總估值與動能指標

用 Gemini 產生 AI 分析報告

整理最近幾季損益表 + 財報亮點 / 風險 / 展望

上傳新聞 / 財報逐字稿 → AI 幫你翻譯 + 摘要

⚠ 本工具僅供學習與研究使用，所有內容皆非投資建議。

🔧 專案功能總覽
1. 股價 & 即時概況

近一小時（或最近收盤價）價格變化

相對漲跌幅（%，正負號標示）

最近一個交易日：

MA5 / MA10 / MA20

成交量

2. 專業圖表（Pro Chart）

K 線圖 / 收盤價折線圖 切換

可選擇顯示：

MA5 / MA10 / MA20

內建 畫線工具：

手畫趨勢線（黑 / 紅 / 藍 / 綠）

Undo / Redo / Clear 多步還原

額外技術指標：

RSI(14)

成交量長條圖

3. 指標摘要卡片

系統會自動計算並顯示：

估值相關

現價

本益比（Trailing PE）

預估本益比（Forward PE）

股價淨值比（P/B）

動能相關

1M / 3M 報酬率

3M 波動度

3M 高點 / 低點

4. 🤖 AI 數據分析（Google Gemini）

使用者可以：

直接看系統預設分析（針對目前圖表區間）

在「追問 AI」中輸入自己的問題（例：
請分析 2024 年整體的估值變化與風險）

AI 會結合：

目前顯示的時間區間 (3mo / 6mo / 1y)

估值 + 動能指標

輸出一份結構化報告，包含：

估值概況（偏高 / 偏低 / 大致合理）

近期股價動能

亮點（2–4 點）

風險與需關注的重點（2–4 點）

給一般投資人的提醒（非投資建議）

若沒設定 GOOGLE_API_KEY，會自動退回規則版（非 LLM），並在文字中明確標示。

5. 📊 財報亮點 / 風險 / 展望

透過 Yahoo Finance 拿到最近幾季的財報資料

根據：

Earnings table（若有）

季損益表（Income Statement Quarterly）

由 Gemini 整理：

財報亮點

潛在風險

管理階層可能關注的重點

給投資人的提醒（非建議）

沒有 LLM 或資料不足時，會顯示規則版提示，而不是亂編。

6. 📄 文字檔上傳 & AI 摘要 / 翻譯

支援上傳：

txt

md

pdf（透過 pdfplumber 解析）

或直接貼上文字內容（例：新聞、財報逐字稿、法說會內容）

系統會先檢查：

文字內容是否 真的與當前股票相關（用 ticker / 公司名關鍵字比對）

通過檢查後：

逐段英文 → 繁中翻譯

整體內容 → AI 摘要（主題 / 亮點 / 風險 / 展望 / 提醒）

🧠 AI 模型選擇（Google Gemini）

側邊欄可選擇：

⚡ Gemini 2.0 Flash（快速）

回應速度快、適合一般互動與多輪追問

🧠 Gemini 1.5 Pro（深度推理）

推理與摘要較強，適合長文本與較複雜的分析問題

兩個選項都會使用同一個 GOOGLE_API_KEY，僅模型 ID 不同。

└── README.md                  # 專案說明

💡 使用流程簡述

輸入股票代號（例：AAPL）

選擇資料區間（3mo / 6mo / 1y）

側邊欄選擇 Gemini 模型（Flash / 1.5 Pro）

按下「開始分析」

瀏覽：

即時價 / 均線 / 成交量

K 線圖 + 畫線工具

指標摘要

AI 數據分析報告

財報亮點 / 風險 / 展望

如有進階問題：

在「追問 AI」輸入你的疑問

例如：「請比較最近三個季度的獲利穩定度」

如有文字檔（新聞 / 財報）：

上傳檔案或貼上文字

系統會幫你翻譯 + 摘要
